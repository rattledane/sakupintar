# SakuPintar

SakupPintar adalah aplikasi sederhana untuk mencatat transaksi keuangan pribadi (pemasukan dan pengeluaran) dengan antarmuka yang bersih dan modern. Proyek ini dibangun menggunakan **HTML**, **CSS**, dan **JS**. Seluruh data disimpan di **localStorage**, sehingga tetap aman meskipun halaman direfresh.
## Fitur Utama

* Tambah transaksi (income atau expense)
* Hapus transaksi
* Total saldo yang diperbarui otomatis
* Dashboard
* Riwayat transaksi
* Tabungan
* Budgeting
* AI Chatbot
* Analisis chart 
* Desain UI minimalis dan terstruktur
* Data tersimpan di localStorage
* Dan lain lain

## Teknologi yang Digunakan

* **React** untuk membangun UI berbasis komponen
* **Vite** sebagai development bundler yang cepat
* **pnpm** untuk manajemen package
* **CSS** untuk styling manual

## Cara Menjalankan Proyek

1. Pastikan sudah menginstal **Node.js** dan **pnpm**.
3. Clone repository ini:

   ```bash
   git clone <repo-url>
   ```
4. Masuk ke folder proyek:

   ```bash
   cd sakuppintar
   ```
5. Install dependencies:
Pastikan pnpm terinstall, jika tidak terinstall jalankan: npm install -g pnpm
Jalankan: pnpm install

6. Mulai proyek
Mulai pengembangan server dengan menjalankan: pnpm run dev

7. Buka di browser:
Pilih yang localhost dengan control + click
   ```
   http://localhost:5173
   ```
